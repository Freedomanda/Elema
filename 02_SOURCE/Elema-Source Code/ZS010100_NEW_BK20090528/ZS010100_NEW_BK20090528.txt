APPEND LF_TVKBZ TO LT_TVKBZ2.

*---ADD BY DMC 2008/05/07 BEGIN

ON   B~EBELN  = C~EBELN

FOR ALL ENTRIES IN LT_TVKBZ2
WHERE A~AEDAT IN PR_DATE
AND B~KNTTP <> 'M'
AND B~LOEKZ = ''
AND B~WERKS = LT_TVKBZ2-VKBUR.

*2008/01/15 ADD END
