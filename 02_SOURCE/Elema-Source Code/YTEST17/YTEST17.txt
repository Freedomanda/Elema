REPORT  YTEST16.

DATA:
LW_ACTIVE1 TYPE C,
LW_ACTIVE2 TYPE C.

PARAMETERS:
P_A RADIOBUTTON GROUP G1 USER-COMMAND SS,
P_B RADIOBUTTON GROUP G1.

PARAMETERS P_FILEP TYPE FILEPATH-PATHINTERN MODIF ID M2.
PARAMETERS P_FILEF TYPE FILE_TABLE-FILENAME MODIF ID M1.
INITIALIZATION.
LW_ACTIVE1 = 1.
LW_ACTIVE2 = 0.
AT SELECTION-SCREEN.
IF SY-UCOMM = 'SS'.
LW_ACTIVE1 = 0.
LW_ACTIVE2 = 1.
ENDIF.

AT SELECTION-SCREEN OUTPUT.
LOOP AT SCREEN.
*   CSVファイル名 変更
CASE SCREEN-GROUP1.
WHEN 'M1'.
SCREEN-ACTIVE = LW_ACTIVE1.
MODIFY SCREEN.
WHEN 'M2'.
SCREEN-ACTIVE = LW_ACTIVE2.
MODIFY SCREEN.
WHEN OTHERS.
ENDCASE.
ENDLOOP.
START-OF-SELECTION.

DO 3 TIMES.
IF SY-SUBRC <> 0.
EXIT.
ENDIF.
ENDDO.
