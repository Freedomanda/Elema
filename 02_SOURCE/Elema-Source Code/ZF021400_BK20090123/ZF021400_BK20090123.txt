C_WA_OUTPUT-ZENGETSUMI_GOKEYI = C_WA_OUTPUT-ZENGETSUMI_SYOGO

ENDAT.

*&---------------------------------------------------------------------*
*&      Form  FILTER_LIFNR
*&---------------------------------------------------------------------*
*       対象絞込処理
*----------------------------------------------------------------------*
*  -->  U_LIFNR        ヘッダ編集用構造
*  -->  C_OK_FLG       対象ﾌﾗｸﾞ(対象となるﾌﾗｸﾞ⇒'T')
*----------------------------------------------------------------------*
FORM FILTER_LIFNR USING    U_WA_HEADER  TYPE STRT_HEADER
CHANGING C_OK_FLG     LIKE C_OK.
DATA: LF_SAGAKU2 TYPE C.   "ﾌﾗｸﾞ:差額②がゼロ以外
DATA: LF_SAGAKU  TYPE C.   "ﾌﾗｸﾞ:残高差額がゼロ以外
DATA: LV_ZERO    TYPE BSEG-DMBTR.

DATA: L_WA_OUTPUT_TMP   TYPE STRT_OUTPUT.

CLEAR: C_OK_FLG.
* 同一仕入先を読み出して　差額②と残高差額の判断を行う
LOOP AT ITAB_OUTPUT INTO L_WA_OUTPUT_TMP
WHERE BUKRS =  U_WA_HEADER-BUKRS AND
LIFNR =  U_WA_HEADER-LIFNR.

** 何れかの通貨で差額が発生する場合、差額フラグをセット
*   残高差額
IF L_WA_OUTPUT_TMP-SAGAKU <> LV_ZERO.
LF_SAGAKU = 'X'.
ENDIF.
*   差額②
IF L_WA_OUTPUT_TMP-SAGAKU2 <> LV_ZERO.
LF_SAGAKU2 = 'X'.
ENDIF.
*   両方とも差額フラグが立ったらループを抜ける
IF LF_SAGAKU = 'X' AND LF_SAGAKU2 = 'X'.
EXIT.
ENDIF.
ENDLOOP.

CASE P_ZOUTYP.
**  出力区分1:差額②、残高差額共に発生する仕入先を出力する
WHEN C_DOUTYP_1.
*     差額②、残高差額、両方の値がゼロ以外の場合
IF LF_SAGAKU2 = 'X' AND LF_SAGAKU = 'X'.
*       対象フラグのセット
C_OK_FLG = C_OK.
ENDIF.
**  出力区分2:差額②、残高差額の何れかが発生する仕入先を出力する
WHEN C_DOUTYP_2.
*     差額②、残高差額、何れかの値がゼロ以外の場合
IF LF_SAGAKU2 = 'X' OR LF_SAGAKU = 'X'.
*       対象フラグのセット
C_OK_FLG = C_OK.
ENDIF.
**  出力区分3:差額②のみ発生する仕入先を出力する
WHEN C_DOUTYP_3.
*     差額②の値がゼロ以外、残高差額がゼロの場合
IF LF_SAGAKU2 = 'X' AND LF_SAGAKU <> 'X'.
*       対象フラグのセット
C_OK_FLG = C_OK.
ENDIF.
**  出力区分4:残高差額のみ発生する仕入先を出力する
WHEN C_DOUTYP_4.
*     差額②の値がゼロ、残高差額がゼロ以外の場合
IF LF_SAGAKU2 <> 'X' AND LF_SAGAKU = 'X'.
*       対象フラグのセット
C_OK_FLG = C_OK.
ENDIF.
*   insert 2008/06/25 S.MIKAMI {
**  出力区分5:差額の発生する仕入先を出力する
WHEN C_DOUTYP_5.
*     差額②の値がゼロの場合
IF LF_SAGAKU2 = 'X'.
*       対象フラグのセット
C_OK_FLG = C_OK.
ENDIF.
**  出力区分6:残高差額の発生する仕入先を出力する
WHEN C_DOUTYP_6.

IF LF_SAGAKU = 'X'.

*&      Form  SET_LAST_GOUKEYI_DATA
*&---------------------------------------------------------------------*
*       画面の差額②と差額データ行の作成処理
*----------------------------------------------------------------------*
FORM SET_LAST_GOUKEYI_DATA.
DATA: L_WA_HEADER  TYPE STRT_HEADER,
L_WA_OUTPUT_TMP       TYPE STRT_OUTPUT,
L_WA_SAGAKU2_GOUKEYI  TYPE STRT_KOUKEYI,    "差額②合計構造
L_WA_SAGAKU_GOUKEYI   TYPE STRT_KOUKEYI,    "差額合計構造
L_SAGAKU2_GOUKEYI     TYPE ZF2020-WRBTR_H,  "差額②合計金額
L_SAGAKU_GOUKEYI      TYPE ZF2020-WRBTR_H.  "差額合計金額
* 対象以外データの削除
DELETE ITAB_OUTPUT WHERE OBJFLG <> C_OK.
L_WA_OUTPUT_TMP-LIFNR = SPACE.
MODIFY ITAB_OUTPUT FROM L_WA_OUTPUT_TMP TRANSPORTING LIFNR
WHERE OBJFLG = C_OK.
SORT ITAB_OUTPUT ASCENDING BY BUKRS LIFNR WAERS.
LOOP AT ITAB_OUTPUT INTO L_WA_OUTPUT_TMP.
*   差額②の合計
L_SAGAKU2_GOUKEYI = L_SAGAKU2_GOUKEYI + L_WA_OUTPUT_TMP-SAGAKU2.
*   差額の合計
L_SAGAKU_GOUKEYI  = L_SAGAKU_GOUKEYI + L_WA_OUTPUT_TMP-SAGAKU.
*   通貨コード
AT END OF WAERS.
*     差額②合計内部テーブルに格納する
L_WA_SAGAKU2_GOUKEYI-WAERS = L_WA_OUTPUT_TMP-WAERS.
L_WA_SAGAKU2_GOUKEYI-GOUKEYI = L_SAGAKU2_GOUKEYI.
APPEND L_WA_SAGAKU2_GOUKEYI TO ITAB_SAGAKU2_GOUKEYI.
*     差額合計内部テーブルに格納する
L_WA_SAGAKU_GOUKEYI-WAERS = L_WA_OUTPUT_TMP-WAERS.
L_WA_SAGAKU_GOUKEYI-GOUKEYI = L_SAGAKU_GOUKEYI.
APPEND L_WA_SAGAKU_GOUKEYI TO ITAB_SAGAKU_GOUKEYI.
CLEAR: L_WA_SAGAKU2_GOUKEYI, L_WA_SAGAKU_GOUKEYI,
L_SAGAKU2_GOUKEYI, L_SAGAKU_GOUKEYI.
ENDAT.
ENDLOOP.
ENDFORM.        "SET_LAST_GOUKEYI_DATA
