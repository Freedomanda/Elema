*&---------------------------------------------------------------------*
*&  Include           MZEGSDR1020I01
*&---------------------------------------------------------------------*
*&---------------------------------------------------------------------*
*&      AT SELECTION-SCREEN
*&---------------------------------------------------------------------*
*       選択画面の入力チェック
*----------------------------------------------------------------------*
AT SELECTION-SCREEN.
* A-1．初期処理
PERFORM init_process_1000.

*&---------------------------------------------------------------------*
*&      Module  USER_COMMAND_9000  INPUT
*&---------------------------------------------------------------------*
*       選択画面9000の輸入後の処理
*----------------------------------------------------------------------*
MODULE user_command_9000 INPUT.
w_save_ok = ok_code.

CLEAR: ok_code.
CALL SCREEN 3100 STARTING AT 11 1
ENDING AT 118 25.
*  CALL SELECTION-SCREEN 3101 STARTING AT 15 8.
IF w_save_ok = cns_exec.
PERFORM display_scr2000.
ENDIF.
ENDMODULE.                 " USER_COMMAND_9000  INPUT

*&---------------------------------------------------------------------*
*&      Module  EXIT  INPUT
*&---------------------------------------------------------------------*
*       選択画面1000の機能キーの処理
*----------------------------------------------------------------------*
MODULE exit INPUT.
w_save_ok = ok_code.

CLEAR: ok_code.

IF w_save_ok = cns_bak1000 OR
w_save_ok = cns_ext1000 OR
w_save_ok = cns_cal1000.
LEAVE PROGRAM.
ENDIF.
ENDMODULE.                 " EXIT  INPUT
*&---------------------------------------------------------------------*
*&      Module  USER_COMMAND_3000  INPUT
*&---------------------------------------------------------------------*
*       text
*----------------------------------------------------------------------*
MODULE USER_COMMAND_3000 INPUT.
LEAVE to SCREEN 3100.
ENDMODULE.                 " USER_COMMAND_3000  INPUT
